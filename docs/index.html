kinopub source plugin for Lampa

Плагин-источник видео kinopub для приложения Lampa. Ориентирован на работу в Smart TV Samsung (Tizen), также работает на webOS, Android и в браузере. Поддерживает встроенный плеер Lampa и Tizen AVPlayer.

Прямая ссылка для добавления в Lampa:
https://mainsync-afk.github.io/lampa_kinopub/kp.js

Установка

  1. В Lampa откройте Настройки → Расширения.
  2. Нажмите Добавить плагин.
  3. Введите URL: https://mainsync-afk.github.io/lampa_kinopub/kp.js
  4. Сохраните и перезапустите Lampa.

Авторизация

  1. Откройте карточку любого фильма или сериала и нажмите кнопку KinoPub.
  2. На экране появится 5-значный код. Откройте на телефоне или ПК kino.pub/device и введите этот код.
  3. После подтверждения плагин получит токен и автоматически продолжит работу.

Настройки

В Настройки → KinoPub доступны:

  • URL лог-сервера — для отладки. Если пусто — логи только в browser console.
  • Макс. качество — потолок битрейта при выборе потока (480p…2160p).
  • Формат потока — HTTP (mp4), HLS, HLS v2/v4, Авто. На Tizen надёжнее HLS.
  • CORS-прокси — необязательно. Заполняйте только если упёрлись в CORS.
  • Авторизоваться / Выйти — управление сессией kinopub.

Удалённые логи

Из-за того что встроенная консоль Lampa на ТВ неудобна для отладки, плагин умеет слать логи на свой простой Node.js-сервер. 

git clone https://github.com/mainsync-afk/lampa_kinopub
cd lampa_kinopub/log-server
node server.js

В настройках плагина указать http://<ip-вашего-пк>:8088 — и логи плагина начнут падать в stdout сервера и в файл logs/YYYY-MM-DD.log.

Сборка

Плагин — один файл kp.js, без транспиляции. Любые правки достаточно закоммитить — GitHub Pages раздаст обновлённую версию по той же ссылке (с учётом кэша браузера/ТВ).

Поддерживаемые платформы

  • Samsung Tizen — основная цель.
  • LG webOS
  • Android (Lampa + любые Android-плееры через Player.runas('android'))
  • Браузер (Lampa Web)

Лицензия и оговорка

Плагин не аффилирован с kinopub. Используется публичный xbmc-клиент Device Flow OAuth, известный сообществу неофициальных клиентов. Подписка на kinopub — ответственность пользователя.
