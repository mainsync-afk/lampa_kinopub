# kp.js log server

Принимает удалённые логи от плагина `kp.js` на Smart TV.

## Запуск

```bash
cd log-server
node server.js
```

По умолчанию слушает `:8088`. Откройте на ПК в браузере `http://<ip-пк>:8088/tail` чтобы увидеть последние 100 записей. Файлы пишутся в `logs/YYYY-MM-DD.log` построчно (JSON Lines).

## Конфигурация

| Переменная  | Значение                  |
|-------------|---------------------------|
| `PORT`      | порт, default 8088        |
| `LOG_DIR`   | директория, default `./logs` |
| `LOG_TOKEN` | если задан — плагин должен слать `X-Log-Token: <token>` |

## Настройка плагина

В Lampa: «Настройки» → «kinopub» → «Лог-сервер» → ввести `http://<ip-пк>:8088`. Плагин шлёт батчами раз в 2 секунды.

## Как смотреть логи в реальном времени

Сервер пишет цветной поток в stdout. Удобный «follow»:

```bash
# Linux/macOS
tail -f logs/$(date +%F).log | jq -c '.'

# Windows PowerShell
Get-Content -Path "logs\$(Get-Date -Format yyyy-MM-dd).log" -Wait
```

## Безопасность

Лог-сервер не предназначен для публичного доступа: пускайте его в локальной сети или закрывайте `LOG_TOKEN`.
